# Report a security issue

If you discover a security vulnerability in VaultPeer Desktop, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, use one of the following:

1. **GitHub Security Advisories** — open a [private security advisory](https://github.com/mHamzaIqbal1998/VaultPeer-Desktop/security/advisories/new) on this repository.
2. **Email** — contact the maintainers through the email listed on the repository profile or release page.

We will acknowledge your report within 5 working days. Coordination and disclosure will be handled through GitHub Security Advisories when possible.

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Affected versions
- Any proof-of-concept or suggested fix, if available
