# VaultPeer Desktop -- Keyboard Shortcuts

A complete reference of keyboard shortcuts available in VaultPeer Desktop. All in-app shortcuts are customizable via **Settings > App > Keyboard Shortcuts**. Global system hotkeys (auto-type) remain fixed.

---

## Global Shortcuts (System-Wide)

These shortcuts work even when VaultPeer is not focused. They are registered as global hotkeys with the operating system and are not customizable.

| Shortcut | Action |
|----------|--------|
| Ctrl+Alt+A | Auto-type credentials into the focused window |
| Ctrl+Alt+P | Selective auto-type (password only) |

---

## Application Shortcuts

### General

| Shortcut | Action |
|----------|--------|
| Ctrl+K | Open global search |
| Ctrl+L | Lock the database |
| Ctrl+S | Save the database |
| Ctrl+N | Create a new entry |
| Ctrl+G | Open the password generator |
| Ctrl+, | Open settings |
| Escape | Close the current overlay, modal, or dialog |

### Entry Actions

These shortcuts apply when an entry is selected in the entry list. If a text input or text selection is focused, standard clipboard behavior takes precedence.

| Shortcut | Action |
|----------|--------|
| Ctrl+C | Copy the selected entry's password |
| Ctrl+B | Copy the selected entry's username |
| Delete | Move the selected entry to the recycle bin |
| Enter | Open the selected entry's detail view |

### Navigation

| Shortcut | Action |
|----------|--------|
| Tab | Move focus to the next pane (sidebar, entry list, detail) |
| Shift+Tab | Move focus to the previous pane |
| Up Arrow | Move selection up in a list or tree |
| Down Arrow | Move selection down in a list or tree |
| Left Arrow | Collapse a group node in the sidebar tree |
| Right Arrow | Expand a group node in the sidebar tree |
| Enter | Select or activate the focused item |
| Space | Select or activate the focused item |

### Search Modal

| Shortcut | Action |
|----------|--------|
| Ctrl+K | Open / focus the search field |
| Up Arrow | Previous search result |
| Down Arrow | Next search result |
| Enter | Navigate to the selected search result |
| Escape | Close the search overlay |

---

## Customization

All in-app shortcuts (everything except the global auto-type hotkeys) can be reassigned:

1. Open **Settings** (Ctrl+,).
2. Go to the **App** tab.
3. Scroll to **Keyboard Shortcuts**.
4. Click on any shortcut binding to capture a new key combination.
5. The new binding takes effect immediately.

Conflicting bindings are detected and flagged. Reset individual shortcuts or restore all defaults from the same panel.
